# Idea behind book_to
book_to არის საიტი განკუთვნილი უპირველეს ყოვლისა პატარა კერძო ბიზნესებისთვის, როგორიცაა კოსმეტოლოგი, კერძო მასწავლებელი და ა.შ.
book_to არის ვებსაიტი სადაც მომხმარებელს აქვს საშუალება როგორც სერვისის შეთავაზების,
ასევე სხვისი სერვისის მოხმარების. შესაძლებელია რეგისტრაცია(რომელიც მეილის ვერიფიკაციას მოიცავს) /დალოგინება ასევე პაროლის აღდგენა.
შესაძლებელია პროფილის განახლება, ფოტოს ატვირთვა და about me ნაწილის რედაქტირება.
ჩვეული სოციალური ქსელის მსგავსად შესაძლებებლია სხვა მომხმარებლის დაფოლოვება და მათი შეთავაზებული სერვისების ნახვა. პირად პროფილს რაც 
შეეხება შესაძლებელია როგორც შეთავაზებული სერვისების ნახვა, ასევე
რეგისტრირებულისაც. ის კი, რაც გამოარჩევს book_to-ს ყველა მისი მეტოქისგან არის შეუდარებელი კალენდრის სისტემა: მომხმარებელს არ უწევს 
გადარეკვა-გადმორეკვა დროის შესათანხმებლად და შემდეგ დროის დასამახსოვრებლად
ფურცელზე დაწერა. არამედ სერვისის შემთავაზებელს აქვს კალენდარი, რომელზეც შეუძლია მონიშნოს დრო როდესაც სცალი/არ სცალია
და ამის შედეგად შეუძლია მომხმარებელს იხელმძღვანელოს. მომხმარებლის მიღება შესაძლებელია მეილის სახით და ავტომატურად
აისახება პროფილის პირად კალენდარზე, როგორც სერვისის შემთავაზებლისთვის, ასევე მომხმარებლისთვის (კალენდარი აქვს ყველა სერვისს 
საკუთარი და ასევე ყველა პროფილს). ცხადია ასეთ
საიტს ესაჭიროება ნოტიფიკაციების ფუნქცია (რომელიც მაგალითად გაცნობებთ თუ ვინ დაგაფოლოვათ/დაგაანფოლოვათ) და ძებნის ფუნქცია.
შესაძლებელია როგორც კატეგორიით ან სახელით ძებნა, ასევე მაგალითად ფასისა და დროით გაფილტვრა. სერვისის ხარისხის შეფასება
შესაძლებელია სტარებით, ხოლო საინტერესო სერვისების შენახვა შესაძლებელია დაგულებით (რომლითაც ძებნაც შეიძლება). ამიტომ
შემოგვიერთდით! დაიწყეთ თქვენი საოცნებო freelance სამსახური დღეს!
